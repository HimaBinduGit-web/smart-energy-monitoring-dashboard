# Smart Energy Monitoring and Analytics Dashboard

This project is a simple energy monitoring dashboard built using Streamlit.

## Features

* Synthetic smart meter data generation
* Energy usage visualization
* Power consumption analytics dashboard

## Technologies Used

* Python
* Streamlit
* Pandas
* Matplotlib

## How to Run

1. Generate data:

```
python data_generator.py
```

2. Run dashboard:

```
streamlit run app.py
```
